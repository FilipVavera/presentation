Inspector automation

Sencha QA demo



Created by Filip Vavera <filip.vavera@profiq.com> and Petr Večeřa <petr.vecera@profiq.com>

What is goal?

  1. To have automated tests for Inspector
  2. Automatically test as much as possible
  3. Run tests on all supported platforms
  4. ...speed up regression testing process!

Sencha Inspector

  1. Allow us to Inspect ExtJS / Touch Apps
    • Component overview
    • Event viewer
    • Layout Runs
    • Stores explorer
    • App details and App Architecture
  2. Brings special features such as
    • SWAM Secure SQL DB Explorer
    • ExtJS 6 live theming
  3. Can be run on Windows, OS X, Linux (planned)
  4. Can inspect apps in all supported browsers by ExtJS

How is it developed

  • Developed in python 2.7 - works on every system
  • Developed with Selenium framework
  • Using python library unittest
  • Separate package for framework and tests
  • Every test is standalone and can be run separately
    • Delete settings and license file
    • We can connect various ExtJS apps
    • We can simultaneously control both
      inspector and inspected app
